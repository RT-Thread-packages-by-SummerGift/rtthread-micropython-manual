# **gc** – 控制垃圾回收

!!! tip "简介"
    `gc` 模块提供了垃圾收集器的控制接口。

## 函数

- gc.enable()  
  启动自动垃圾回收。

- gc.disable()  
  禁用自动垃圾回收。 堆内存仍然可以分配，垃圾回收仍然可以手动启动使用 gc.collect().

- gc.collect()  
  运行垃圾回收。

- gc.mem_alloc()  
  返回分配的堆RAM的字节数。

- gc.mem_free()  
  返回可用堆内存的字节数。
  
更多内容可参考  [gc](https://docs.python.org/3.5/library/gc.html#module-gc) 。
